RESUMEN PADDING CSS


Propiedad de Padding (relleno) Las propiedades de
relleno de CSS definen el espacio entre el elemento
de borde y el elemento de contenido. Se puede definir
el valor de relleno de la

siguiente manera:

padding-top: 10px;
padding-right: 10px;
padding-bottom: 10px;
padding-left: 10px;
