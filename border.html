NORMAS DE LAS REGLAS DE ESTILO

Las propiedades de borde de CSS le permiten especificar el estilo y el color del borde del elemento.

border-width (anchura de borde) La propiedad border-width es utilizada para configurar el ancho de borde. El ancho es especificado en píxeles, o por medio de uno de los tres valores predefinidos: thin, medium, o thick.

border-color (color de borde) La propiedad border-width es utilizada para configurar el color de borde. Se puede configurar color por:

name – specify a color name, like “red”
RGB – specify a RGB value, like “rgb(255,0,0)”
Hex – specify a hex value, like “#ff0000”
border-style (estilo de borde)
